Smart Gate Keeper

[Tizen IoT Team #5]

사물 인식 기반 문지기

Team Members: 권오승, 김준비, 김태홍


Description

  • Captures images continuously with a USB camera
  • Detects a valid object in the images
  • Controls servomotors based on image analysis
  • Sends the images to connected web clients
  • Displays a live view of images in each web client
